# Chrome Web Store Copy

## Privacy Policy

PDF Text to Speech processes text extracted from PDF files so it can generate spoken audio for the user.

What the extension does:

- Reads text from PDF files opened in the browser or uploaded by the user.
- Sends extracted text to pdftext2speech.com to process text-to-speech requests.
- Uses OpenAI to generate audio from the submitted text.
- Stores a device token and limited billing state to manage free usage, subscriptions, and purchases.

What data is processed:

- PDF text selected for reading aloud.
- A device token used to identify usage limits and subscription state.
- Billing-related identifiers when a user starts or manages a paid plan through Stripe.
- Support request details if the user contacts support.

What data is not sold:

- We do not sell personal data.

How data is used:

- To generate speech audio for requested PDF content.
- To enforce the free usage limit and paid subscription rules.
- To process payments and subscription access through Stripe.
- To respond to support requests.

Third parties:

- OpenAI for text-to-speech generation.
- Stripe for checkout, billing, and subscription management.

Data retention:

- Device and billing-related records may be retained as needed to provide service access, enforce usage limits, prevent abuse, and maintain subscription state.

Contact:

- support@pdftext2speech.com

## Store Listing

### Short Description

Read PDF files aloud in Chrome with natural AI voice playback.

### Detailed Description

PDF Text to Speech reads PDF documents aloud directly in Chrome.

Use it to:

- listen to PDF files in the browser
- upload a PDF and start playback quickly
- control playback speed
- continue with paid listening after the free trial

How it works:

- The extension extracts text from the PDF you open or upload.
- That text is sent to pdftext2speech.com for speech generation.
- Audio is generated using OpenAI voice models.

Free and paid usage:

- Includes 5 free minutes of listening.
- After the free limit is used, playback requires a paid plan.
- Payments and subscriptions are handled through Stripe.

Privacy summary:

- PDF text is processed only to generate audio playback.
- A device token is used to track free usage and subscription state.
- Payment processing is handled by Stripe.

### Suggested Privacy Disclosure

This extension sends extracted PDF text to pdftext2speech.com and OpenAI to generate speech audio. It also stores a device token to manage free usage and subscription access.
